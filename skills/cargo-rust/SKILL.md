## Core Concept

Cargo manages Rust projects with:
- Dependency resolution
- Build orchestration  
- Testing and benchmarking
- Publishing to crates.io

## Commands

```bash
# Build
cargo build
cargo build --release

# Test
cargo test
cargo test --lib

# Check (fast)
cargo check
cargo clippy

# Run
cargo run
cargo run --release

# Add dependency
cargo add serde
cargo add serde --features derive
```
