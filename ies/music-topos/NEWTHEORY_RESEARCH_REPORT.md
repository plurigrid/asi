## 5. Technical Deep Dive: Bispectral Neural Networks

### 5.1 Core Innovation

**Problem:** Learning group-invariant representations that are **complete** (preserve all non-group-related information)

**Solution:** Bispectrum - an analytically defined group invariant

**Key Properties:**
- Invariance to compact commutative group actions
- Completeness: preserves all signal structure
- Strong adversarial robustness
- Can learn groups, representations, and invariant maps from data

### 5.2 Mathematical Foundation

**Bispectrum Definition:**
For signal f and group G, the bispectrum captures:
- All Fourier components
- Their phase relationships
- Group-invariant structure

**Relationship to:**
- Lie group theory
- Harmonic analysis
- Representation theory
- Information geometry

### 5.3 Applications

**Theoretical:**
- Complete invariants for image recognition
- Provable adversarial robustness
- Learning group structure from symmetries

**Practical:**
- Visual recognition tasks
- Scientific computing
- World model learning (New Theory's focus)

## 6. Technical Deep Dive: Moose Framework

### 6.1 Architecture

**Core Components:**
1. **Compiler** - Translates computations to dataflow graphs
2. **Runtime** - Executes distributed computation
3. **Python eDSL** - High-level programming interface
4. **Rust API** - Low-level performance primitives

**Key Features:**
- Placement-aware computation
- Replicated secret sharing (RSS)
- Support for fixed-point arithmetic
- Production-ready (created at Cape Privacy)

### 6.2 Security Model

**MPC via Replicated Secret Sharing:**
- 3-party honest-majority protocol
- Information-theoretic security
- Efficient for ML operations

**Threat Model:**
- Honest-but-curious adversaries
- Collusion resistance (1 out of 3)
- Network security via gRPC

### 6.3 Use Cases

**Current:**
- Privacy-preserving machine learning
- Secure data processing
- Federated analytics

## Appendix A: Technical Glossary

**Bispectrum:** Complete group-invariant representation preserving all non-group signal structure

**Group Equivariance:** Property where transforming input and transforming output are equivalent

**Lie Group:** Smooth manifold with group structure; continuous symmetry transformations

**Replicated Secret Sharing (RSS):** MPC protocol using redundant shares across parties

**Geomstats:** Python package for Riemannian geometry and machine learning on manifolds

**Moose:** Secure distributed dataflow framework for encrypted ML/data processing
